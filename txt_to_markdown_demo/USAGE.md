# txt-to-Markdown Demo Usage

## Files

- `sample_input.txt` — example plain-text input
- `sample_output.md` — expected Markdown output
- `converter.py` — CLI converter script

## Run

```bash
python3 converter.py sample_input.txt
```

By default, the converter writes `sample_input.md` next to the input file.

## Notes

This demo shows how the converter turns simple text structure into Markdown headings, lists, quotes, and fenced code blocks.
